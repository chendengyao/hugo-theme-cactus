MongoDB安装使用

2022-10-10
3 minute read
数据库 , MongoDB , NoSQL

环境:

Ubuntu 20.04.1 LTS (GNU/Linux 5.4.0-47-generic x86_64)

安装

MongoDB有社区版和企业版，社区版就行了。找到 Ubuntu安装的官方文档。 安装命令如下:

1# 添加公钥
2wget -qO - https://www.mongodb.org/static/pgp/server-3.6.asc | sudo apt-key add -
3# 添加软件源，不同的Ubuntu版本可能会不同，建议看官方文档
4echo "deb [ arch=amd64 ] https://repo.mongodb.org/apt/ubuntu bionic/mongodb-org/3.6 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-3.6.list
5
6sudo apt-get update
7# 安装
8sudo apt-get install -y mongodb-org

安装成功查看MongoDB版本：

启动

MongoDB的配置文件在 /etc/mongod.conf 里面，先简单介绍一下：

# 官方的配置文档：
#   http://docs.mongodb.org/manual/reference/configuration-options/
# 存放数据的地方
storage:
  dbPath: /var/lib/mongodb
  journal:
    enabled: true
# 日志存放的地方
systemLog:
  destination: file
  logAppend: true
  path: /var/log/mongodb/mongod.log
# 运行的端口和允许访问的ip
net:
  port: 27017
  bindIp: 127.0.0.1

如果要修改运行的端口之类的可以先在配置文件里面修改之后再启动。

 1# 启动
 2sudo systemctl start mongod
 3# 查看运行状态
 4sudo systemctl status mongod
 5# 自启动
 6sudo systemctl enable mongod
 7# 停止运行
 8sudo systemctl stop mongod
 9# 重启
10sudo systemctl restart mongod

如果启动失败的话可以查看日志 /var/log/mongodb 寻找相关解决方法。

连接和简单使用

MongoDB 也提供了shell工具来操作数据库：

 1# 进入shell，默认本机27017端口
 2mongo
 3# 指定ip和端口连接MongoDB
 4mongo --host 127.0.0.1:27017
 5# 显示当前使用的数据库，默认的数据库为test
 6db
 7# 显示全部数据库
 8show dbs
 9# 切换当前操作的数据库，没有则会直接创建
10use dbName
11# 在数据库插入一条数据,这里的user在MongoDB称之为Collection相当于Mysql表的概念，即在user表中添加一条数据
12db.user.insertOne({name:"cdy",sex:"male",age:23,food:["apple","banana"]})
13# 查看Collection中所有的数据
14db.user.find()
15# 列出所有的Collection
16show collections
